# monitoring
GUI of the overall system

##Requirements
**JavaScript:**
* Leaflet 0.7.7
* jQuery 2.2.3

**Python (v3):**
* Flask 0.10.1
* tqdm 4.7.4
* utm 0.4.0
* Shapely 1.5.16
* numpy 1.8.2
* matplotlib 1.5.0


##Usage
1. run WebServer:

```bash
python3 master/scripts/webserver/webserver.py
```

2. open browser
3. go to: http://127.0.0.1:5000/
